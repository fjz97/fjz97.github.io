    public static <T> ArrayList<T> concat(Stream<ArrayList<T>> stream) &#123;
        Optional<ArrayList<T>> optional = stream.reduce(Test::arrayListConcat);
        return optional.get();
    &#125;

    public static <T> ArrayList<T> arrayListConcat(ArrayList<T> prev, ArrayList<T> next) &#123;
        ArrayList<T> temp = (ArrayList<T>) prev.clone();
        temp.addAll(next);
        return temp;
    &#125;

注意reduce（聚合）和collect（收集）的区别，就是让Stream<ArrayList>转换成ArrayList和ArrayList<ArrayList>的区别。
